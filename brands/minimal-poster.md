# Brand DNA: Minimal Poster

## Identity

- **ID**：`minimal-poster`
- **定位**：为快速传播设计的高对比海报系统，用少量元素制造清晰记忆点。
- **适用**：社交媒体卡片、观点海报、系列封面、活动预告、演示章节页。
- **不适用**：复杂后台、长篇阅读、需要大量交互说明的工具页。
- **关键词**：直接、锐利、节制、强节奏、可识别。

## Visual Character

一张海报只表达一个核心动作或观点。视觉张力来自比例、裁切、网格和文字，而不是装饰堆叠。系列内容通过稳定母版与可变主视觉形成统一。

## Color

| Token | Value | Semantic role |
|---|---|---|
| `--bg` | `#F5F4EF` | 浅色主底 |
| `--surface` | `#FFFFFF` | 图片与留白面 |
| `--ink` | `#111111` | 主标题与结构 |
| `--ink-inverse` | `#F7F6F2` | 深底反白 |
| `--line` | `#111111` | 明确分割 |
| `--accent` | `#D9473F` | 传播焦点 |
| `--secondary` | `#2855C7` | 系列区分或数据对照 |

- 单张海报只选择一个主强调色；第二强调色只用于系列编码或比较。
- 黑白/纸色约 80%，强调色约 20%。
- 不使用渐变；色块必须承担分区、编号或焦点功能。

## Typography

- 中文标题：`Source Han Sans SC` / `Noto Sans SC`，700-900 字重。
- 中文辅助：同家族常规字重；短引语可使用 `Source Han Serif SC`。
- 英文展示：`Arial Narrow` / `Helvetica Neue Condensed`。
- 主标题通常 2-5 行，允许有控制的断行和裁切，但不得损害关键词识别。
- 不用 viewport 宽度直接缩放字体；使用固定断点与 `clamp()` 保证容器内适配。

## Graphic Language

- 使用严格网格、粗细明确的分割线、大号文字、实心矩形、裁切图片和序号。
- 图标只在表达动作时出现；避免一组装饰性小图标。
- 同系列固定边距、署名区和页码位置，主构图可变化。
- 圆角 0-8px；不用柔软胶囊和漂浮阴影。

## Imagery

- 一张卡最多一个主要图像焦点，优先人物、对象或真实结果。
- 允许高对比黑白、双色调和大胆裁切；脸部与产品关键信息不能误裁。
- 插画使用平面几何、丝网印刷或强轮廓风格。
- 不使用模糊氛围图、无主体 stock photo 和无关纹理。

## Motion

- 适合快速切入、遮罩揭示、整块位移和按节拍分页，160-280ms。
- 交互反馈应干脆，不使用弹性、漂浮和长时间渐变。
- 静态导出必须保留完整信息，不依赖动画才能理解。

## Decoration

允许序号、系列名、短线、色块、裁切文字和注册标记。装饰总数应少，且与网格对齐。

## Prohibitions

- 多层卡片、柔和阴影、玻璃质感、复杂渐变。
- 一张海报放多个同权重观点。
- 小字过多、正文段落过长、无控制的自动换行。
- 把 emoji 当主视觉，或用多个贴纸填空。
- 为“极简”牺牲必要来源、署名和可读性。

## Recommendations

- **Components**：`quote-block`、`badge`、`annotation`、`card`。
- **Layouts**：`hero`、`comparison`、`grid`、`two-column`。
- 原因：它们支持单一焦点、强对比和系列化编排。

## Acceptance

- 缩略图尺寸下仍能读出核心观点。
- 一张只存在一个第一视觉焦点。
- 所有小字在目标导出尺寸下可读。
- 系列页面共享稳定母版，但主构图不机械重复。
- 没有依赖阴影、渐变或无意义贴纸制造层次。
