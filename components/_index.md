# Component Index

Component 是可复用语义零件。读取组件文件后，再读取当前 Brand 的适配规则。

| ID | 用途 | 文件 |
|---|---|---|
| `card` | 聚合一个完整信息对象 | `components/card.md` |
| `terminal-window` | 命令与运行结果 | `components/terminal-window.md` |
| `prompt-block` | AI 输入、变量与输出契约 | `components/prompt-block.md` |
| `quote-block` | 引用或核心观点 | `components/quote-block.md` |
| `annotation` | 指向具体内容的解释 | `components/annotation.md` |
| `badge` | 状态、类型、版本 | `components/badge.md` |
| `timeline` | 时间事件集合 | `components/timeline.md` |
| `workflow-node` | 流程中的输入、处理或输出 | `components/workflow-node.md` |
| `code-block` | 可阅读和复制的代码 | `components/code-block.md` |

旧版 HTML/CSS 示例仍在 `references/components.md`，仅在需要实现细节时选读，不再作为组件分类模型。
