# Layout: Grid

- **适合**：同类功能、案例、资源、作品或可筛选对象。
- **槽位**：`grid-heading`、可选 `filter`、重复 `item`、可选 `pagination`。
- **信息顺序**：先定义集合，再按稳定规则排序；项目内部字段顺序一致。
- **视觉特点**：网格用于比较，不等于所有内容都做成卡片；允许图片、文字或混合条目。
- **响应式**：使用明确的最小列宽与 `auto-fit`；避免内容变化导致卡片跳动。
- **不适用**：存在强顺序、因果或叙事关系的内容。
- **可用 Scene**：`landing`、`app`、`social-card`、`presentation`。
