# Audience: General Readers

- **背景**：对主题有兴趣，但不假设专业知识。
- **目标**：在较短时间内理解“是什么、为什么重要、下一步做什么”。
- **阻力**：术语、信息过密、缺乏上下文、看不到实际例子。

## Adaptation

- 首次出现的术语用一句自然语言解释，不堆缩写。
- 每个 section 只承担一个问题；段落控制在 2-4 句。
- 先给结果和例子，再补机制；复杂流程控制在 3-5 步。
- 使用真实案例、前后对照和明确结论建立可信度。
- Card 与 Annotation 可承担解释；Terminal 和 Code Block 只保留必要行。
- 正文字号不低于 16px，关键操作不能只依赖颜色。

成功标准：非专业读者能复述核心概念，并知道下一步行动。
