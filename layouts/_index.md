# Layout Index

Layout 只组织信息，不携带 Brand 样式。

| ID | 主要用途 | 文件 |
|---|---|---|
| `hero` | 建立对象、价值与第一行动 | `layouts/hero.md` |
| `two-column` | 解释与证据、主次并置 | `layouts/two-column.md` |
| `step-flow` | 顺序、依赖、操作流程 | `layouts/step-flow.md` |
| `comparison` | 差异、选择、前后对照 | `layouts/comparison.md` |
| `timeline` | 时间、演进、里程碑 | `layouts/timeline.md` |
| `grid` | 同类项目扫描与比较 | `layouts/grid.md` |
| `dashboard` | 高频任务、状态与控制 | `layouts/dashboard.md` |

选择顺序：先根据 Scene 的信息结构选择，再用 Audience 调整密度，最后用 Brand 渲染视觉。
