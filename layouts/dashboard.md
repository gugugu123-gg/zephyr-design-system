# Layout: Dashboard

- **适合**：后台、监控、工作台、多对象操作和高频决策。
- **槽位**：`global-nav`、`context-header`、`primary-workspace`、`summary`、`detail-panel`、`feedback-layer`。
- **信息顺序**：范围与上下文先于数据；高频任务占据主区域，低频设置后置。
- **视觉特点**：稳定栅格、明确层级、紧凑但可扫描；状态与动作靠近对象。
- **响应式**：小屏保留核心任务，辅助面板转为抽屉或独立视图；不把所有列机械压缩。
- **不适用**：营销介绍页、观点海报、单线叙事。
- **可用 Scene**：`app`，少量用于 `tutorial` 的交互演示。
