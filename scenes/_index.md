# Scene Index

Scene 定义内容目标和信息结构，所有 Brand 共用。

| ID | 交付类型 | 文件 | 常用模板 |
|---|---|---|---|
| `tutorial` | 教程、概念解释、操作指南 | `scenes/tutorial.md` | `assets/template-tutorial.html` |
| `landing` | 产品、活动、项目介绍页 | `scenes/landing.md` | `assets/template-landing.html` |
| `app` | 工具、后台、工作台 | `scenes/app.md` | `assets/template-app.html` |
| `social-card` | 社交卡片、观点组图 | `scenes/social-card.md` | `assets/template-cards.html` |
| `article` | 长文、公众号、专题阅读 | `scenes/article.md` | `assets/template-wechat.html` |
| `presentation` | 演示文稿、分享会、提案 | `scenes/presentation.md` | 无固定模板，按目标媒介生成 |

Scene 决定 section 顺序，Brand 决定视觉语言，Audience 决定表达深度。
