# Design Styles Index — 设计风格总表

> 使用 Skill 时，AI 必须先把这里的主要风格方向展示给用户选择。用户不选时，默认使用「暖纸编辑感」。

| 风格 | 文件 | 适合内容 | 排版关键词 | 色彩规则 |
|------|------|----------|------------|----------|
| 暖纸编辑感 | `styles/warm-editorial.md` | 科普解释、教程、观点文章、个人品牌页 | 大标题、强留白、非对称、杂志感 | 灰暖纸张背景 + 黑字 + 极少量编辑红/黄色 |
| 极简竖版观点卡 | `styles/vertical-poster-cards.md` | 抖音/小红书卡片、观点封面、知识解释、系列首屏 | 巨大标题、强留白、顶部小标签、底部短说明 | 纸张底 + 黑字 + 单一小面积编辑红或黄色 |

## 给用户的选择话术

开始设计前，先给用户看这个简版菜单：

1. **暖纸编辑感** — 灰暖纸张背景、大标题、克制高亮，适合科普/教程/观点内容。
2. **极简竖版观点卡** — 巨大中文标题、强留白、少量标签/页脚，适合抖音/小红书卡片和观点解释。
3. **后续可扩展** — 如果用户给了参考图或明确风格，把它登记成新的 style 文件。

如果用户只说“按我的风格来”，默认选 **暖纸编辑感**，但仍需确认交付形态和排版类型。
